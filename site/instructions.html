Ellsworth American Printing Service FTP Instructions

First, unless you have an existing FTP client, you'll need to download FileZilla.
http://sourceforge.net/projects/filezilla/
Click Download Filezilla on the page.
Click on Filezilla Client.
If you're running Windows, click on the green box with the ".exe" in it. Otherwise, you'll need to download a client executable for your own operating system.
Download and install FileZilla.
Open FileZilla and create a site in the Site Manager (the icon just under "File"). You can also access it via the File menu itself, or by pressing CTRL+S.
Click on New Site, type in a name for the site, and fill out the following information:

HOST: ftp.eaprintingservice.com
PORT: 21
SERVER TYPE: FTP
LOGON TYPE: Normal
For Username and Password information, please contact Matt Martin.
When you're finished, just click "Connect".

To copy files to our server, you can drag the files in into the box in the lower right hand corner.
